# Social Media Multi-Agent System (CrewAI)

This project automates a 30-day social media content pipeline using multiple CrewAI agents.  
It includes:

- Content creation agent  
- Hashtag research agent  
- Scheduling agent  
- Analytics agent  

## How to run

python src/main.py


## Output

- `outputs/content_calendar.json`
- `outputs/analytics_report.json`

## Technologies
- Python 3.8+
- CrewAI
